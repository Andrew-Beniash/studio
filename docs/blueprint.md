# **App Name**: Profile Header

## Core Features:

- Header Component: Display a header at the top of the page with a background color.
- User Avatar: Display the user's avatar on the right side of the header.
- Profile Tool: LLM selects appropriate header and user avatar image for the particular user (requires user profile info).

## Style Guidelines:

- Background color for the header: Desaturated light blue (#EAF4F8).
- Primary color: A serene blue (#3498db) is chosen for its calming effect and wide recognition as a color associated with trust and professionalism. Its familiarity in tech interfaces makes it easily approachable for users.
- Accent color: Complementary vivid blue (#2980b9) is employed for interactive elements to ensure visual distinction.
- Font: 'Inter' (sans-serif) for a clean and modern look in headers and user information.
- Header: Fixed position at the top of the page, spanning the full width.
- Avatar: Circular, positioned to the far right of the header, with a small margin from the edge.
